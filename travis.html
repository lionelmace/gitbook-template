Integration with Travis Pipeline

Integration with Travis Pipeline

  1. Go to the Settings on your profile in https://travis-ci.org/account/repositories

  2. Hit Sync account if you do not see the repo in the list

  3. Activate the repo

  4. Go to the Settings of the project and add the environment variable GITHUB_TOKEN
